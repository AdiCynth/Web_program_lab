# Web Program Lab Work
Bloom & Petal — Online Flower Shop 🌸
A multi-page static website built with HTML and CSS as part of the Web Programming Lab (WPL) Experiment 1. The project simulates an online flower shop called Bloom & Petal, featuring a responsive homepage with featured floral arrangements, a product catalogue displayed in a structured table with pricing and availability, a user login page with form validation, and a registration page collecting customer details including delivery address. An additional academic timetable page demonstrates advanced table layouts using colspan and rowspan. Styled with a modern, clean design system using CSS custom properties, grid layouts, and hover animations.
Tech Stack: HTML5, CSS3
